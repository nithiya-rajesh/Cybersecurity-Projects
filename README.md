# 🛡️ Cybersecurity Projects Portfolio
### Integrated Security Suite: Offensive (Red Team) & Defensive (Blue Team) Tooling

This repository serves as a comprehensive portfolio of my work in cybersecurity. It features custom-built tools and scripts designed to simulate attacks, automate defenses, and perform digital forensics. The repository is divided into two main domains: **Offensive Security** for penetration testing and **Defensive Security** for monitoring and incident response.

---

## 🔴 Offensive Security (Red Team)
*Tools designed for reconnaissance, exploitation, and vulnerability assessment.*

| Project | Description | Primary Language |
| :--- | :--- | :--- |
| 🐍 **SQL-Viper** | Advanced SQL injection scanner and automated exploitation helper. | Python |
| 👻 **Ghost-Scanner** | Stealthy network reconnaissance tool designed to bypass basic IDS/IPS detection. | Bash / Python |
| ⚡ **AuthStorm** | A multi-threaded authentication stress-tester for web portals and services. | Python |
| 🔍 **Vuln-Scanner** | Automated vulnerability scanner mapping endpoints against known CVE databases. | Python |
| 🔗 **Red-Connect** | A lightweight C2 (Command & Control) framework for reverse shell management. | Go / C++ |
| 🔑 **Password-Spray-Tester** | Simulates password spraying attacks to identify weak organizational credentials. | Python |

## 🔵 Defensive Security (Blue Team)
*Tools designed for threat detection, network monitoring, and digital forensics.*

| Project | Description | Primary Category |
